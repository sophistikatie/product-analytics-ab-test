# Product Analytics Case: A/B Test Simulation (E-commerce)

## Business Context

E-commerce платформа.  
Проблема: низкая конверсия пользователей из `view → purchase`.

Baseline conversion: **8.2%**

Гипотеза: улучшение карточки товара увеличит конверсию на **5% (относительно)**.

Цель анализа — смоделировать A/B-тест и оценить статистическую значимость и потенциальный бизнес-эффект.

---

##  Data Validation

- Проанализировано **1 000 000 пользовательских событий**
- Проверены типы данных, пропуски и распределение событий
- Проверено отсутствие критичных дубликатов
- Выделены ключевые события: `view`, `cart`, `purchase`

---

## Funnel Analysis

Построена воронка:

Baseline conversion rate: **8.2%**

Это означает, что ~91.8% пользователей, просмотревших товар, не совершают покупку.

---

## A/B Test Simulation

Поскольку реальный онлайн-эксперимент не проводился, была выполнена симуляция A/B-теста:

- Пользователи, совершившие `view`, случайным образом разделены 50/50
- Единица анализа — уникальный пользователь
- В тестовой группе смоделирован относительный рост конверсии +5%
- Проведён двухвыборочный z-test

**p-value = 0.0018**  
Различие статистически значимо (p < 0.05).

---

## Business Impact

Рост конверсии на ~0.42 процентных пункта приводит к:

- ≈ 732 дополнительным покупкам на аудитории 177k пользователей
- Потенциальному приросту выручки:
  - ~227k (по среднему чеку)
  - ~130k (по медианному чеку)

Изменение может иметь существенный финансовый эффект.

---

## Data Mart

Создана пользовательская витрина данных, агрегирующая события на уровне пользователя:

- `viewed`
- `purchased`
- `revenue`

Витрина позволяет корректно рассчитывать конверсию и выручку без работы с сырыми событиями и может использоваться для дальнейшей аналитики и построения BI-дашбордов.

---

## Limitations

- Эксперимент является симуляцией на исторических данных
- Не учитывалась сезонность
- Не проводилась сегментация пользователей
- Не рассчитывался минимальный обнаруживаемый эффект (MDE)
- Не анализировались guardrail-метрики

Для принятия финального решения рекомендуется проведение реального онлайн A/B-теста.
